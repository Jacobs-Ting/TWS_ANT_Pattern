## 📖 專案簡介 (Overview)

本專案是一個基於瀏覽器運行的 **3D 射頻物理現象視覺化模擬器 (Proof of Concept)**。
針對 TWS (True Wireless Stereo) 藍牙耳機在人體配戴時常見的極端惡劣環境，本工具將深奧的微波工程現象（如近場介電質吸收、阻抗失調、場型塌陷）轉化為高互動性、視覺化的 3D 動態模型。

旨在打破 RF 硬體工程師與 PM、軟體團隊及客戶之間的溝通壁壘，提供一個無須依賴昂貴商業電磁模擬軟體（如 HFSS / CST）即可即時展示系統級天線效能的溝通平台。

## ✨ 核心功能 (Key Features)

* **動態距離模擬 (Dynamic Distance Simulation)**
  透過 UI 滑桿實時控制耳機從「自由空間 (Free Space)」逼近「人體表面 (On-Head)」的過程，無縫演繹電磁場的物理變化。
* **純數學合成場型 (Synthetic Anisotropic Pattern)**
  為確保 100% 的商業 IP 合規與機密保護，本 PoC **未匯入任何真實測量或模擬數據**。輻射場型（包含 LDS PIFA 參考地破碎產生的多瓣特徵）完全由 WebGL / GLSL Shader 透過空間雜訊與數學幾何即時運算生成。
* **近場耦合與場型塌陷 (Near-field Coupling & Collapse)**
  在 Vertex Shader 中實作嚴格的世界座標碰撞邊界。當電磁波進入人頭 2.5cm 近場無功區時，精確模擬能量被高介電常數組織吸收的「深藍色熱力圖渲染」與「高頻相位扭曲凹陷」。
* **即時 RF Link Budget 計算**
  內建連動的物理數學近似模型，動態計算並展示：
  * **$\eta$ (輻射效率)**：隨距離劇烈衰減。
  * **S11 (Return Loss)**：透過 Chart.js 即時繪製共振頻偏（Detuning）曲線，展示頻率從 2.5 GHz 偏移至 2.45 GHz 且深度劣化的過程。
  * **TRP (總輻射功率)**：基於 Class 2 (2.0 dBm) 傳導功率即時換算。

## 🛠️ 技術棧 (Tech Stack)

* **Frontend:** HTML5, Vanilla JavaScript, CSS3
* **3D Rendering:** [Three.js](https://threejs.org/) (WebGL)
* **Shader Programming:** GLSL (Custom Vertex & Fragment Shaders)
* **Data Visualization:** [Chart.js](https://www.chartjs.org/)

本專案為純前端架構，無須複雜的 Node.js 編譯環境或伺服器部署。

1. Clone 此儲存庫：
   ```bash

使用瀏覽器（推薦 Google Chrome 或 Edge）直接開啟資料夾內的 index.html 即可運行。
